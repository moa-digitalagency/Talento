## Presentation de la plateforme

taalentio.com est une plateforme professionnelle de centralisation des talents africains. Elle permet aux professionnels de creer des profils detailles, de valoriser leurs competences et de se connecter a des opportunites dans differents secteurs.

### Les deux modules principaux

**Module Talents Generaux**
Pour tous les professionnels : techniciens, artisans, createurs, commerciaux, etc. Ce module permet de referencer vos competences dans plus de 150 categories de metiers.

**Module CINEMA**
Dedie a l'industrie audiovisuelle : acteurs, figurants, cascadeurs, mannequins, etc. Ce module offre des fonctionnalites specifiques pour le casting et la gestion des productions.

## Inscription et creation de compte

### Inscription Talents Generaux

1. **Accedez a la page d'inscription**
   - Cliquez sur "Inscription Talent" depuis la page d'accueil
   - Le formulaire s'affiche en plusieurs sections

2. **Section Identite**
   - Renseignez votre prenom et nom
   - Date de naissance
   - Genre (Homme/Femme/Non precise)
   - Numero de passeport ou carte de sejour (optionnel, donnees chiffrees)

3. **Section Contact**
   - Adresse email valide (vous recevrez vos identifiants par email)
   - Numero de telephone avec indicatif
   - WhatsApp (optionnel)
   - Adresse postale

4. **Section Localisation**
   - Pays d'origine
   - Ville d'origine
   - Nationalite
   - Pays de residence
   - Ville de residence

5. **Section Competences**
   - Selectionnez vos talents parmi les categories proposees
   - Vous pouvez choisir plusieurs talents
   - Indiquez vos annees d'experience

6. **Section Disponibilite**
   - Temps plein, temps partiel, flexible, etc.
   - Mode de travail prefere : sur site, teletravail, hybride
   - Tarification souhaitee (horaire et/ou mensuel)

7. **Section Langues**
   - Cochez les langues que vous parlez

8. **Section Documents**
   - Photo de profil (formats PNG, JPG - max 5 Mo)
   - CV (formats PDF, DOC, DOCX - max 10 Mo)

9. **Section Liens**
   - Portfolio, site web personnel
   - Reseaux sociaux : LinkedIn, Instagram, Twitter, etc.

10. **Validation**
    - Apres soumission, vous recevez un email avec :
      - Votre code unique (ex: MAM0001RAB)
      - Un mot de passe temporaire
    - Conservez bien ces informations

### Inscription CINEMA

L'inscription au module CINEMA suit un processus similaire avec des champs specifiques :

1. **Caracteristiques physiques**
   - Taille en cm
   - Couleur des yeux (12 options)
   - Couleur de cheveux (16 options)
   - Type de cheveux (10 options)
   - Teint de peau (10 options)
   - Morphologie (10 options)

2. **Types de talents cinema**
   - Acteur/Actrice Principal(e)
   - Acteur/Actrice Secondaire
   - Figurant(e)
   - Silhouette
   - Doublure / Doublure Lumiere
   - Cascadeur/Cascadeuse
   - Mannequin
   - Voix Off
   - Choriste / Danseur

3. **Competences additionnelles**
   - Arts martiaux
   - Danse
   - Sports
   - Instruments de musique
   - Autres talents artistiques

4. **Productions precedentes**
   - Historique de vos participations

## Connexion a votre espace

### Methodes de connexion

Vous pouvez vous connecter de deux manieres :

**Avec votre email**
- Entrez votre adresse email
- Entrez votre mot de passe

**Avec votre code unique**
- Entrez votre code (ex: MAM0001RAB)
- Entrez votre mot de passe

### Premiere connexion

Lors de votre premiere connexion :
1. Utilisez le mot de passe temporaire recu par email
2. Vous serez invite a le modifier
3. Choisissez un mot de passe securise (minimum 8 caracteres)

### Mot de passe oublie

Contactez l'administrateur avec votre code unique ou email pour recevoir un nouveau mot de passe.

## Votre profil personnel

### Tableau de bord

Apres connexion, vous accedez a votre tableau de bord qui affiche :
- Votre photo de profil
- Votre code unique avec QR code
- Vos informations principales
- Votre score de profil

### Modifier votre profil

1. Cliquez sur "Modifier le profil"
2. Mettez a jour les informations souhaitees
3. Cliquez sur "Enregistrer"

### Elements modifiables

- Informations personnelles
- Photo de profil
- CV
- Competences et talents
- Reseaux sociaux
- Bio professionnelle
- Disponibilite et tarifs

### Score de profil

Votre score de profil (0-100) est calcule selon :
- Completude des informations personnelles
- Presence d'une photo
- Presence d'un CV
- Nombre de talents declares
- Liens sociaux renseignes
- Qualite du CV (analyse IA)

### QR Code personnel

Chaque profil dispose d'un QR code unique qui :
- Permet un acces rapide a votre profil public
- Peut etre telecharge et imprime
- Facilite le partage de vos coordonnees professionnelles

## Module CINEMA

### Acces au module

Le module CINEMA est accessible depuis :
- La page d'accueil ("Inscription Cinema")
- Le menu principal pour les utilisateurs connectes

### Profil CINEMA

Votre profil CINEMA comprend :
- **Photo de profil** haute resolution
- **Galerie photos** (plusieurs photos sous differents angles)
- **Caracteristiques physiques** completes
- **Competences artistiques** detaillees
- **Historique de productions**

### Code unique CINEMA

Format : PPVVVNNNNNNG (12 caracteres)
- PP : Code pays (ex: MA pour Maroc)
- VVV : Code ville (ex: CAS pour Casablanca)
- NNNNNN : Numero sequentiel
- G : Genre (M/F)

Exemple : MACAS000001F

### Etre selectionne pour un projet

Quand vous etes selectionne :
1. Vous recevez un email de notification
2. L'email contient les details du projet
3. Un badge personnalise est genere
4. Vous pouvez telecharger votre badge en PDF

### Gestion des presences

Si vous etes assigne a un projet :
- Votre presence est enregistree chaque jour
- Heure d'arrivee et de depart
- Duree de travail calculee automatiquement
- Historique consultable

## Questions frequentes

### Mon profil

**Comment modifier ma photo ?**
Allez dans "Modifier le profil", section "Photo", et telechargez une nouvelle image.

**Quels formats de CV sont acceptes ?**
PDF, DOC et DOCX, avec une taille maximale de 10 Mo.

**Comment ajouter des talents ?**
Dans "Modifier le profil", section "Competences", cochez les nouveaux talents.

### Connexion

**J'ai oublie mon mot de passe**
Contactez l'administrateur avec votre email ou code unique.

**Je ne recois pas mes emails**
Verifiez votre dossier spam. Si le probleme persiste, contactez l'administrateur.

### CINEMA

**Comment apparaitre dans les castings ?**
Completez votre profil au maximum, ajoutez des photos de qualite et renseignez toutes vos competences.

**Qu'est-ce que le matching IA ?**
Un systeme qui analyse votre profil et le compare aux criteres des productions pour vous proposer des opportunites correspondantes.

### Technique

**Le site ne fonctionne pas**
Videz le cache de votre navigateur et rafraichissez la page.

**Ma photo ne s'affiche pas**
Verifiez que l'image est au format PNG ou JPG et fait moins de 5 Mo.
